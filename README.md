# 💬 Mood Chatbot - AI Personality Based Chat Assistant

A fun and interactive chatbot application that lets users choose the mood/personality of their AI companion before starting a conversation.

Instead of a normal chatbot, this project allows you to chat with different AI personalities — **Angry Bot, Flirty Bot, and Funny Bot** — each responding with its own unique tone and style.

The idea behind this project was to explore how Large Language Models can be customized to generate different conversational experiences using prompt engineering.

---

## ✨ Features

- 🎭 **Multiple AI Personalities**
  - 😡 Angry Bot — replies with an irritated, sarcastic, and bold personality
  - 😉 Flirty Bot — responds in a playful and charming tone
  - 😂 Funny Bot — keeps conversations humorous and entertaining

- 💬 Interactive chat interface built with Streamlit

- 🧠 LLM-powered responses using prompt-based personality customization

- ⚡ Real-time conversational experience

- 🎨 Simple and user-friendly UI

---

## 🚀 How It Works

1. User selects a chatbot personality:
   - Angry Bot
   - Flirty Bot
   - Funny Bot

2. Based on the selected personality, a customized system prompt is provided to the Large Language Model.

3. The LLM generates responses while maintaining the selected tone throughout the conversation.

4. Streamlit provides the interactive interface where users can chat with their selected AI companion.

## 🎯 Project Motivation

This project was built as a fun exploration of Generative AI and how changing prompts can completely transform the behavior and personality of an AI assistant.

It demonstrates how LLM applications can be customized beyond simple question-answer chatbots by creating engaging and personalized user experiences.
